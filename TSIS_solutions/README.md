# TSIS Solutions

Solved tasks for `programming-principles-2/TSIS`.

## How to run

Install dependencies:

```bash
pip install -r requirements.txt
```

### TSIS1 PhoneBook
1. Create PostgreSQL database.
2. Edit `TSIS1/config.py`.
3. Run SQL files or let the app initialize schema from menu:
```bash
python TSIS1/phonebook.py
```

### TSIS2 Paint
```bash
python TSIS2/paint.py
```

### TSIS3 Racer
```bash
python TSIS3/main.py
```

### TSIS4 Snake
1. Edit `TSIS4/config.py` for PostgreSQL.
2. The game also works in local JSON fallback mode if DB is unavailable.
```bash
python TSIS4/main.py
```
