# Practice 09 — Pygame Games

Готовое решение для **Practice 09** из репозитория.

## Что внутри

- `mickeys_clock` — часы с минутной и секундной стрелками в стиле Mickey
- `music_player` — музыкальный плеер с управлением с клавиатуры
- `moving_ball` — игра с красным шаром и ограничением границ
- `requirements.txt` — зависимость `pygame`

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

### Mickey's Clock
```bash
cd mickeys_clock
python main.py
```

### Music Player
```bash
cd music_player
python main.py
```

### Moving Ball
```bash
cd moving_ball
python main.py
```
