# Mickey's Clock

Приложение показывает текущие **минуты и секунды** в реальном времени.

## Запуск

```bash
python main.py
```

## Управление

- `Q` — выход
- `X` на окне — закрыть приложение

## Что реализовано

- синхронизация с системным временем
- обновление каждую секунду
- правая рука — минутная
- левая рука — секундная
- использована картинка `images/mickeyclock.jpeg`
