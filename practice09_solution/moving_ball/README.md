# Moving Ball

Игра с красным шаром размером 50x50 (радиус 25).

## Запуск

```bash
python main.py
```

## Управление

- `↑ ↓ ← →` — движение шара
- `Q` — выход

## Что реализовано

- белый фон
- красный шар
- перемещение на 20 пикселей за одно нажатие
- шар не выходит за границы экрана
- некорректное движение за границу игнорируется
