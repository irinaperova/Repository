# Music Player

Музыкальный плеер на `pygame.mixer` с управлением с клавиатуры.

## Запуск

```bash
python main.py
```

## Управление

- `P` — play
- `SPACE` — pause
- `S` — stop
- `N` — next track
- `B` — previous track
- `Q` — quit

## Что реализовано

- воспроизведение нескольких треков
- переключение вперёд и назад
- отображение текущего трека
- полоска прогресса
- автоматический переход к следующему треку после завершения
